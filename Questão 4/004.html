onmouseover (passe o ponteiro do mouse por cima)

onfocus (selecione o "input" abaixo)

onchange (escolha uma das opções do "select" abaixo)

onsubmit (adicione dados no "input" abaixo e então envie clicando no botão ao lado)

onkeydown (digite uma letra, obs: onkeydown é ativado após uma tecla ser pressionada)

onkeyup (digite uma letra, obs: onkeydown é ativado após uma tecla ser solta)

onmouseout (tire o ponteiro do mouse de cima, obs: o evento onmouseout é ativo após o ponteiro sair do elemento)

ondblclick (clique 2 vezes)

onmousedown (clique no elemento, obs: onmousedown é ativado após o elemento ser pressionado com o mouse)

onmouseup (clique no elemento, obs: onmouseup é ativado após o elemento ser solto)
